# NS Películas - Website Oficial

Site institucional da NS Películas, especializada em instalação de películas automotivas, arquitetônicas e de segurança no Distrito Federal.

## 🚀 Sobre o Projeto

Website moderno e responsivo desenvolvido para a NS Películas, apresentando:
- Catálogo de serviços automotivos e arquitetônicos
- Portfólio de trabalhos realizados
- Sistema de agendamento de atendimento
- Design responsivo para todos os dispositivos
- Otimização para SEO

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **Tailwind CSS** - Framework CSS utilitário
- **JavaScript Vanilla** - Interatividade e funcionalidades
- **Font Awesome** - Ícones
- **Google Fonts** - Tipografia (Montserrat & Inter)

## 📁 Estrutura do Projeto

```
ns-peliculas/
├── index.html          # Página principal
├── css/
│   └── styles.css      # Estilos personalizados
├── js/
│   └── scripts.js      # Funcionalidades JavaScript
├── assets/
│   ├── icons/          # Ícones personalizados
│   └── images/         # Imagens do site
└── FOTOS/              # Portfólio de trabalhos
```
